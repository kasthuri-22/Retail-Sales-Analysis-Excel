# Retail Sales Data Analysis (Excel Project)

This project analyzes a retail sales dataset using Microsoft Excel to identify customer purchasing patterns, product performance, and regional sales trends.

## Tools Used

Microsoft Excel

## Techniques Applied

* Data exploration
* Sorting and filtering
* Descriptive statistics
* Frequency distribution
* Histogram analysis
* Pareto analysis
* Pivot tables
* Data visualization

## Key Insights

* Mid-range spending dominates customer purchases.
* A small number of products generate most of the revenue.
* Regional performance differences highlight marketing opportunities.
