Retail Sales Analysis – Excel Data Analytics Case Study
Project Overview

This project analyzes a retail sales dataset to understand customer purchasing behaviour, product performance, and regional sales trends. The goal is to extract insights that can support better business decision-making.

Business Problem

A retail company wants to understand:

Which products generate the most revenue
Customer spending patterns
Regional sales performance
Preferred payment methods

Analytical Approach

The analysis was performed using Microsoft Excel and includes:

Data exploration
Sorting and filtering
Descriptive statistics
Frequency distribution
Histogram analysis
Percentiles and quartiles
Pareto analysis (80/20 rule)
Pivot table analysis
Data visualization
Key Insights
The revenue distribution is right-skewed, indicating that a small number of high-value orders significantly increase total revenue.
Laptop, Phone, and Printer generate nearly 80% of the total revenue.
The North region produces the highest sales.
Most customers fall within the mid-range spending segment.
Business Recommendations
Focus marketing efforts on top-performing products.
Expand promotional campaigns in high-performing regions.
Introduce loyalty programs for high-value customers.
Optimize pricing strategies within the most common spending range.
Tools Used
Microsoft Excel
Pivot Tables
Data Visualization
Descriptive Statistics
Pareto Analysis
Project Outcome

This project demonstrates how Excel can be used to perform exploratory data analysis, generate insights, and build business-focused dashboards.
